任务范围：
1. 从已解析的 attendance import 生成 wage record workbook。
2. 记录 wage record `.xls` 和 task report generated file。
3. 提供下载接口。
4. 不做 Web UI。

说明：
- 如果代码已经存在，本任务改为复核并补齐缺口。
- 不要重写已满足验收的实现。

业务要求：
1. 使用 `samples/wage/20260601-0630_wageRecords.xls` 作为模板。
2. 必须复制模板后写入，不能修改模板。
3. 生成前必须确认 attendance import 已解析；未解析时返回明确错误。
4. 有 parser errors 的 import 默认不得生成正式 wage record，除非业务明确允许带 warning 生成。
5. 生成文件必须记录：
   - file type
   - storage path
   - SHA-256
   - mime type
   - file size
   - generated by user
6. 重复生成必须有清晰策略：生成新记录、supersede 旧记录，或明确覆盖当前记录；不能静默丢历史。
7. 生成失败必须记录失败原因。
8. 权限必须使用 attendance generate/read。

API：
- `POST /api/attendance-imports/:id/generate-wage-record`
- `GET /api/attendance-imports/:id/files`
- `GET /api/attendance-imports/:id/files/:fileId/download`

验收标准：
1. 能从已解析真实打卡表生成 wage record `.xls`。
2. 模板文件未被修改。
3. wage record 文件被记录并可下载。
4. task report 文件被记录并可下载。
5. 未解析 import 不能生成 wage record。
6. 失败有明确 API 错误和 generated file 状态或错误记录。
7. 测试通过。

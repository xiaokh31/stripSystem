Windows 11 内部生产部署结论

可以开始 Windows 11 企业内网生产部署准备，但建议按“条件通过”执行： 在目标 Windows 11 主机完成 Docker Desktop full stack 启动、数据库 migration、 RBAC seed、首个 ADMIN 登录、OFFICE/WAREHOUSE 登录、权限拒绝、备份恢复 dry-run、 真实打印机和真实 PDA/手机扫码验证后，再正式切换给仓库使用。
